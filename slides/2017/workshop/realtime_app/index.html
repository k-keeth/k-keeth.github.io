3. クライアント側でsocket.ioの処理を書く

基本的にSocket.ioでは

  • socket.on('キー')：受信処理
  • socket.emit('キー')：送信処理
となっています。
では、bbs.ejsに次の行を追記してください。（※長いので次のページへ） 

        <script>
           $(function () { 
              // socket.ioのインスタンス生成
              var socket = io()
              $('form').submit((e) => {
                 // サーバー側にデータを送信
                 socket.emit('chat message', e.target.txt.value)
               })
            })
        </script>
