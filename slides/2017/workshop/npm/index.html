# NPMについて
## NPM（Node Package Manager） 『npm is the package manager for JavaScript and the world’s largest software registry. Discover packages of reusable code — and assemble them in powerful new ways.』 （[npm公式より）](https://www.npmjs.com/)
## npmも一つのコマンドです。 ## 試しにnpmコマンドを実行してみましょう！
# npm -v npmのバージョンが出力されたかと思います。
（ちなみに私は`4.2.0`でした）
## では、本格的にnpmの使い方を見ていきましょう！
## npmの基本コマンド ```shell ・ npm install ・ npm list ・ npm uninstall ・ npm update ・ npm search ``` 日常で使うのはこの5つだと思います(ﾟ∀ﾟ)
## First!! ```shell npm install ```
```shell npm install [-g|--save|--save-dev] module ``` ### 意味：モジュールのインストール ```shell （省略形： npm i） ```
## `[-g|--save|--save-dev]`の意味は？
## こうです。 ```css -g: グローバルでインストールするもの --save: 実際のアプリ稼働時に利用するもの --save-dev: 開発時に利用するもの ```
## 実際に`lite-server`というモジュールを
インストールしてみましょう♪
```shell $ npm install -g lite-server ``` 又は ```shell $ npm i -g lite-server ```
## Next!! ```shell npm list ```
```shell npm list [-g|--depth=n] (module) ``` ### 意味：インストールされたモジュールの一覧表示 ```shell （省略形： npm ls） ```
### `lite-server`がインストールされたか確認してみましょう！
以下のコマンドを実行して下さい♪ ```shell $ npm list -g [--depth=0] ```
### では起動してみましょう！
以下のコマンドを実行して下さい♪ ```shell $ lite-server ```
## Next!! ```shell npm uninstall ```
```shell npm uninstall [-g|--save|--save-dev] module ``` ### 意味：モジュールの削除 ```shell （省略形: npm rm, npm up） ```
### 先程インストールした`lite-server`を削除してみましょう！
以下のコマンドを実行して下さい♪ ```shell $ npm uninstall -g lite-server ```
### `lite-server`が削除されたか確認してみましょう！ ```shell $ npm list -g [--depth=0] ```
## Next!! ```shell npm update ```
```shell npm update [-g|--save|--save-dev] (module) ``` ### 意味：モジュールのアップデート
※moduleを指定しなければ全モジュールが対象 ```shell （省略形: npm up） ```
## updateは後ほど実際に叩いてもらいますm(_ _)m
## Last!! ```shell npm search ```
```shell npm search module ``` ### 意味：対象モジュールに関連するモジュールの一覧を表示 ```shell （省略形: npm s, npm find） ```
## npmのその他使いそうなコマンド ```shell ・ npm info module 　→ モジュールのバージョン情報を表示 ・ npm init 　→ package.jsonの初期化（後述） ``` これらもたまに使うかと思います( ﾟ∀ﾟ)
