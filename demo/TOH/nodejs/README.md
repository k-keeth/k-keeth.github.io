# TOH by Node.js

This application is a rework of Angular's tutorial [TOUR OF HEORS](https://angular.io/docs/ts/latest/tutorial/) by Node.js.

# Usage

```cmd
$ cd /path_to_this_app/
$ npm i
$ npm start
```

After executing the above command, access to [http://localhost:3000](http://localhost:3000)
